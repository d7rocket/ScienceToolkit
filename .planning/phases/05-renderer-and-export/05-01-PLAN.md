FILE 2: carousel-ui/src/parser/parseMarkdown.ts

Parser logic — extract from the /science markdown format:

  - Title: first line matching /^# (.+)$/m
  - Metadata: line matching /**Date:** YYYY-MM-DD | **Field:** X | **Sources:** N/
  - Slides: each block starting with /^## Slide (\d+):\s*(.+)$/m, body text runs until next ## or ---
  - Color Scheme: section matching /## Color Scheme/ containing lines like:
      - Background: #0B0E2D — midnight
      - Primary text: #F0F0F5 — frost
      - Accent: #6C5CE7 — nebula
      - Highlight: #00CEC9 — teal
  - If Color Scheme section is absent (common case): colors = null
  - Role assignment: slide index 1 = 'hook', slide index equal to total = 'cta', others = 'body'

Export a single function: export function parseMarkdown(text: string): ParsedCarousel

  import { useEffect } from 'react';
  import { DropZone } from './components/DropZone';
  import { MetaBar } from './components/MetaBar';
  import { useCarouselStore } from './store/useCarouselStore';

  export default function App() {

    useEffect(() => {

          <div className="flex-1 flex items-center justify-center text-neutral-600 text-sm">
